🎯 Veradigm Test Generator - Help Guide

📝 How to Generate Tests

  1. Enter test criteria in the text area (e.g., "Login to CE Portal and create a case")
  2. Click "Generate Tests" or press Ctrl+Enter
  3. AI will generate Gherkin scenarios based on your criteria

🚀 How to Execute Tests

  1. Click "Execute Selected Tests" to choose test cases from Jira (XSP-58)
  2. Select test cases and click "Execute Tests"
  3. A Test Execution will be created in Jira
  4. Tests run automatically via GitHub Actions
  5. Results are posted to Slack and updated in Jira

📤 Upload to Jira & GitHub

  • Uploads generated tests to Jira Xray and GitHub repository
  • Creates test cases in XSP project
  • Links tests to Test Plan XSP-58

🔄 Other Features

  • Regenerate: Click to regenerate tests with the same criteria using AI variations
  • Save to Features Folder: Saves generated .feature files to your local features folder
  • Copy to Clipboard: Copies generated Gherkin scenarios to clipboard
  • View Report: Available after test execution completes - shows detailed results with screenshots
