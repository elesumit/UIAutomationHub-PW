This repository contains a Playwright-based automation testing framework designed for scalable and maintainable UI test automation.

The framework supports BDD-style testing, reusable components, data-driven execution, and cross-browser testing to enable efficient automation for modern web applications.

Key Features

Playwright + TypeScript based automation

BDD support using Gherkin feature files

Reusable step definitions

Page Object Model (POM) architecture

Data-driven testing using external test data

Cross-browser testing

Chromium

Edge

WebKit

Parallel test execution

Screenshot and trace capture on failures

Configurable environment support

CI/CD integration ready
